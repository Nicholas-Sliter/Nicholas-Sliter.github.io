Exploring 18th Century Dutch Textile Trade - An Interactive Map

Description

  • Nicholas Sliter

  • GitHub ShinyApps Article

  • January 2021 - June 2021

An extenson of what started as a final project for my data science class in the winter 2021, this project map and visalization tool that allows complicated research queries to be resolved visually and fast. The project includes reactive visualizations of any combination of factors. It includes reactive level filtering and a table output for saving the results of queries.
